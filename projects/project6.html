Face Recognition and Alert System Using YOLO

What Was Done

  • Data Augmentation: Expanded face training dataset via rotations, brightness changes, and blurs.
  • YOLOv5 Model Training: Fine-tuned model to recognize faces accurately in various conditions.
  • Wireless IP Camera Integration: Live real-time monitoring from a remote camera stream.
  • Automated WhatsApp Alerts: Triggered when unknown faces are detected.

What Was Learned

  • Advanced data augmentation strategies for small datasets.
  • YOLOv5 model fine-tuning for edge deployment scenarios.
  • Camera integration and live object detection inference.
  • Real-time alert systems with WhatsApp API for surveillance use cases.

What Was Achieved

Successfully deployed a working face recognition security system using edge AI techniques combined with real-time remote monitoring and alert messaging.
