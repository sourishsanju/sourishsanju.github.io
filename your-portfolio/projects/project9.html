Bicopter Control System

Objective

To design and implement a PID and LQR-based control system for stabilizing a custom bicopter UAV, enabling stable vertical lift, hovering, and simple trajectory tracking.

What Was Done

  • Designed bicopter model and CAD frame using SolidWorks.
  • Implemented PID control for basic pitch/roll/yaw stability in ROS2.
  • Developed LQR control strategies for improved stabilization in dynamic scenarios.
  • Ran simulation-based tuning using Simulink before real-world flight experiments.

Challenges

  • Managing unstable pitch-yaw coupling unique to bicopter dynamics.
  • Real-time gain tuning for LQR controller with physical motor limitations.
  • Maintaining hover stability under varying battery loads.

What Was Learned

  • Real-world validation of model-based control design (PID, LQR).
  • Understanding motor asymmetry effects in minimal rotor systems.
  • Critical role of sensor fusion (IMU + visual feedback) in flight stabilization.

What Was Achieved

Successfully stabilized a minimal bicopter aerial vehicle using combined PID and LQR strategies. Demonstrated consistent vertical lift, hover, and basic trajectory following with real-world validation flights.
